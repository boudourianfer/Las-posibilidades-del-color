# Las-posibilidades-del-color
A menudo la película fotográfica de color sólo se usa para captar imágenes, sin aprovechar al máximo las posibilidades que brinda el propio color. Cuando coinciden varios factores favorables, el color puede ser un buen instrumento para la creación de imágenes con un magnífico cromatismo. La potencia de esta imagen se organiza a partir del rostro femenino central, construido con bloques cromáticos intensos que generan contrastes simultáneos entre tonos cálidos y fríos. La nitidez de la mirada actúa como punto de anclaje visual, mientras las figuras en movimiento, difusas, introducen dinamismo y profundidad. El entorno urbano y la luz refuerzan una atmósfera contemporánea donde color, escala y tránsito equilibran la composición.
